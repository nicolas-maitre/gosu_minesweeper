# Gosu Minesweeper
Minesweeper implementation in ruby using gosu.

## Launch option (new!)
- `-d`: choice of difficulty. [beginner, intermediate, expert]  
  Ex: `./main.exe -d intermediate`
  
- `-s`: Custom grid dimensions  
  Ex: `./main.exe -s 8x12`
  
- `-m`: Custom mines count  
  Ex: `./main.exe -s 5x5 -m 2`
  
- `-z`: Custom zoom level (%)  
  Ex: `./main.exe -z 150`
